**Full Stack User Management System – Task List**

**( Inova It Systems )**

1. Create a Spring Boot project.   
2. Configure PostgreSQL database connection.   
3. Create the User entity.   
4. Create the UserRepository.   
5. Create the Service layer.   
6. Create the Controller layer.   
7. Implement **Register User** API (POST /register).   
8. Implement **Login** API (POST /login).   
9. Implement **Get All Users** API (GET /users).   
10. Implement **Get User by ID** API (GET /users/{id}).   
11. Implement **Update User** API (PUT /users/{id}).   
12. Implement **Delete User** API (DELETE /users/{id}).   
13. Add request validation (required fields, email format, password length).   
14. Implement proper exception handling and error responses.   
15. Test all APIs using Swagger. 

**React Frontend**

16. Create a React project.   
17. Configure Axios for API communication.   
18. Create a Login page.   
19. Create a Register page.   
20. Create a Dashboard/Home page.   
21. Create a User List page.   
22. Display all users in a table.   
23. Add Edit User functionality.   
24. Add Delete User functionality.   
25. Add form validation.   
26. Display success and error messages.   
27. Add navigation between pages using React Router.   
28. Style the application. 

**Optional (Bonus)**

29. Encrypt passwords.   
30. Implement JWT authentication.   
31. Add search functionality for users.   
32. Add pagination to the user list.   
33. Add sorting by name or email.   
34. Add logout functionality.   
35. Push both backend and frontend projects to GitHub with a proper README. 

**Expected Outcome:** A complete full-stack User Management System with Register, Login, User CRUD operations, PostgreSQL database integration, and a React frontend communicating with the Spring Boot backend.
